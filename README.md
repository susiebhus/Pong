# Pong 游戏 - Godot 学习项目

## 项目简介
这是一个使用Godot引擎开发的简单Pong游戏，作为我个人学习Godot游戏开发的学习项目。通过这个项目，我实践了Godot引擎的基础知识，包括场景管理、节点系统、GDScript编程和基本的游戏逻辑实现。仅仅是一个非常粗糙的版本，作为我学习的记录。

## 游戏特性
- 经典的Pong游戏玩法
- 双人对战模式（玩家1 vs 玩家2）
- 简单的计分系统

## 控制方式
- **玩家1 (左侧挡板)**: W (上) / S (下)
- **玩家2 (右侧挡板)**: 上箭头 / 下箭头

## 如何运行
1. 确保已安装Godot引擎（版本4.x）
2. 克隆或下载本项目
3. 用Godot打开项目文件夹
4. 在Godot编辑器中点击"运行"按钮

或者，如果您导出了可执行文件：
1. 下载对应您操作系统的版本（暂时只打包了win和mac）
2. 直接运行可执行文件


## 依赖项
- Godot Engine 4.x
